# Desarrollo Web en Entorno Cliente

Repositorio de actividades y ejercicios prácticos del curso de **Desarrollo Web en Entorno Cliente**, organizados por unidades temáticas.

## 🌐 Ver el proyecto

👉 [https://jairoSanroman.github.io/CursoJairo/](https://jairoSanroman.github.io/CursoJairo/)

---

## 📁 Estructura del repositorio

```
CursoJairo/
├── index.html          # Página principal con todos los enlaces
├── Unidad1/            # HTML: estructura, formato, tablas, listas, enlaces
│   ├── ejercicio1/     # Estructura básica HTML5
│   ├── ejercicio3/     # Formato de texto
│   ├── ejercicio4/     # Fórmulas matemáticas y químicas
│   ├── ejercicio5/     # Texto preformateado
│   ├── ejercicio6/     # Listas
│   ├── ejercicio7/     # Tablas: horario y notas
│   ├── ejercicio8/     # Tabla con celdas expandidas
│   ├── ejercicio9/     # Enlaces y direccionamientos
│   ├── ejercicio10/    # Marcos y capas
│   ├── ejercicio11/    # Capas y posicionamiento
│   └── ejercicio12/    # Mapa web - Clínica Veterinaria
└── Unidad2/            # Imágenes y multimedia
    ├── ejercicio2/     # Imágenes en la Web
    ├── ejercicio3/     # Galería de imágenes
    └── ejercicio4/     # Mapa de imágenes
```

## 📚 Contenido por unidades

### Unidad 1 – HTML: Estructura y contenido

| Ejercicio | Descripción |
|-----------|-------------|
| 1 | Estructura básica de una página HTML5 |
| 2 | Instalación y comparativa de navegadores web *(teórico)* |
| 3 | Etiquetas de formato de texto: negrita, cursiva, subrayado, tachado |
| 4 | Superíndices y subíndices: fórmulas matemáticas y químicas |
| 5 | Texto preformateado con `<pre>` y poema de Bécquer |
| 6 | Listas ordenadas, no ordenadas, de definición y anidadas |
| 7 | Tablas: horario semanal y notas de alumnos |
| 8 | Tabla con celdas expandidas (`colspan` y `rowspan`) |
| 9 | Enlaces internos, externos, a imágenes y a archivos PDF |
| 10 | Proyecto con marcos (`<frame>`) y capas |
| 11 | Capas y posicionamiento con `<div>` |
| 12 | Mapa web de una clínica veterinaria con navegación entre páginas |

### Unidad 2 – Imágenes y multimedia

| Ejercicio | Descripción |
|-----------|-------------|
| 1 | Etiquetas para imágenes, audio, vídeo e incrustación de YouTube *(teórico)* |
| 2 | Inserción de imágenes en la web |
| 3 | Galería de imágenes |
| 4 | Mapa de imágenes con áreas clicables |
